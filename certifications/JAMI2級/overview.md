# JAMI 2級

## ⚠️ 受験までの3段階戦略

JAMI 2級は **受験資格の取得が前提** となるため、以下の3段階で進める：

```
[Phase 1] 受験資格の取得  →  [Phase 2] 試験申込・準備  →  [Phase 3] 試験対策・受験
```

---

## Phase 1: 受験資格の取得（最優先）

- [ ] **単位取得 9単位分**（セミナー受講等）

### 入会・単位取得の段取り
- [ ] JAMI公式サイトで入会手続きを確認
- [ ] 年会費・入会金を確認
- [ ] セミナー開催スケジュールを把握（2026年5月〜2027年初めの間に9単位確保）
- [ ] 単位の有効期限を確認
- [ ] CFP・キャリコンの学習と並行可能なペースに調整

> ※入会・単位取得は時間がかかるため、CFP第1回（6月）終了後、本格着手予定。
> ただし入会手続きは早めに動く（CFP学習と並行可能）。
