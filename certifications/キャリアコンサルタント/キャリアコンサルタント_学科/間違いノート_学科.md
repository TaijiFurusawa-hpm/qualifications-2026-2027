## 第31回（2026/3実施）失点15問

### キャリア理論

**問8** ✗ シュロスバーグ4Sに「Standard（基準）」が含まれると誤選択
→ ◯ 4Sは「**Self / Situation / Support / Strategies**」。Standard（基準）は誤り。
→ ◯ ハンセン（Hansen, L.S.）＝**統合的人生設計（ILP）**：仕事を人生の他役割と統合して捉える。〔GT1〕

### カウンセリング

**問3** ✗ 「社内規程が整備されていなければ活動できない」を正答と誤選択
→ ◯ 社内規程がなくても活動は可能。CCは従業員の**職業生活の設計と能力開発を支援しつつ、組織の活性化につながるという視点**も必要。〔GC1〕

**問10** ✗ 「構成的グループ・エンカウンター、バーン（Berne, E.）、シェアリング」を正答と誤選択
→ ◯ バーン（Berne, E.）＝**交流分析**（TA）の提唱者。構成的グループ・エンカウンターは**國分康孝**。
→ ◯ 正答組み合わせ：**自律訓練法・シュルツ（Schultz, J.H.）・身体の弛緩**。〔GC2〕

**問33** ✗ 「第4段階でショック離職防止」を正答と誤選択
→ ◯ 第3段階（職場復帰**直前**）＝**復帰後のイメージ作りの支援**
→ ◯ 第4段階（職場復帰**後**）＝フォローアップ・環境適応支援。〔GC3〕

**問34** ✗ フィードバックを「カウンセラーの考えや感じたことを伝えること（自己開示）」と誤解
→ ◯ フィードバック（アイビイ）＝「カウンセラーあるいは**第三者が、クライエントをどう見ているかというデータを与えること**」
→ ◯ カウンセラーの考え・感じたことを伝えるのは「**自己開示**」。〔GC4〕

**問36** ✗ 「グループアプローチは健康な人を対象に行われる」を正答と誤選択
→ ◯ グループアプローチは健康な人だけでなく精神疾患等を持つ人も対象。
→ ◯ **SST（ソーシャルスキルトレーニング）＝学習理論に基づく認知行動療法の一つ**（正答）。
→ ◯ 構成的グループ・エンカウンター（GE）＝**國分康孝**が開発（ロジャーズではない）。〔GC5〕

**問40** ✗ 「同一化」の説明として「自分の感情を他人が向けていると捉える」を正答と誤選択
→ ◯ それは「**投影（Projection）**」の説明。
→ ◯ **論理的過誤**＝「あの特徴があるのだからこの特徴もあるはず」（正答）
→ ◯ **ハロー効果**＝一部の特徴・印象で全体を判断する傾向（全体評価への波及）
→ ◯ **寛容効果**＝意識的・無意識的に甘く判断しがちになる傾向（選択肢3・4の説明が入替）。〔GC6〕

**問44** ✗ 「複数の方策を選ばせる」を正答と誤選択
→ ◯ 方策実行の支援では、実行**前**に「**方策の内容・目的・原理・プロセス・結果・利点と損失などをクライエントに説明する**」。
→ ◯ 方策は複数選ばせず、合意した1つを実行する。価値観に反すれば方策は変更する。〔GC7〕

### 法令制度

**問22** ✗ 「無期転換申込みで使用者は正社員として雇用しなければならない」を正答と誤選択
→ ◯ 無期転換後は**正社員になるわけではない**。原則として現在の有期と**同一の労働条件で無期雇用**に転換される（別段の定めがない限り）。
→ ◯ 正答：有期労働契約はやむを得ない事由がなければ**契約期間満了前の途中解雇不可**。〔GL1〕

**問26** ✗ 「タイプ4の取組みについて」の選択肢を不適切と誤選択（実際は正しい記述）
→ ◯ 最も不適切：「タイプ2 キャリア教育も**インターンシップ**と称される」→ 誤り。
→ ◯ インターンシップと称されるのは**タイプ3（汎用型・専門活用型）とタイプ4（高度専門型）のみ**。タイプ2はキャリア教育であってインターンシップではない。〔GL2〕

**問42** ✗ 「トライアル雇用は原則6ヶ月」と誤解
→ ◯ トライアル雇用の試行期間は**原則3ヶ月**（最長6ヶ月）。
→ ◯ 正答：自ら事業を営む・役員で**週30時間以上**就労中の人はトライアル雇用の対象外。
→ ◯ ハローワーク経由の求人が必要（求人を出していない事業主は利用不可）。〔GL3〕

### 統計白書

**問18** ✗ 完全失業率の分母を「15歳以上人口」と誤解
→ ◯ 完全失業率＝**（労働力人口－就業者）÷労働力人口×100**。分母は**労働力人口**。〔GS1〕

**問19** ✗ 有効求人倍率の出典を「毎月勤労統計調査」と誤解
→ ◯ 有効求人倍率は**一般職業紹介状況（職業安定業務統計）**が毎月公表。
→ ◯ 毎月勤労統計調査＝賃金・労働時間・雇用指数（求人倍率は出ない）。〔GS2〕

**問20** ✗ 社会インフラ関連職の賃金について「月額約5万円高い」を正答と誤選択
→ ◯ 正答：社会インフラ関連職は**スキル・経験蓄積に応じた賃金上昇の仕組みが相対的に弱い可能性**がある（これが人材確保困難の背景）。〔GS3〕

**問21** ✗ 「特定技能での外国人労働者数は2023年12月時点で約200万人」を正答と誤選択
→ ◯ 特定技能での就労者数は約200万人ではなく**約20万人程度**（一桁違い）。
→ ◯ 正答：「外国人は月額賃金が高い求人・年間休日が多い求人に応募しているが、**残業時間が多い求人にも多く応募している**」。〔GS4〕

## 第30回（2025/11実施）失点12問

### 統計白書

**問2** ✗ キャリアに関する相談先として「職場の上司・管理者」が最多と知らず選択肢4を選択
→ ◯ 令和6年度能力開発基本調査：**正社員・正社員以外ともにキャリア相談先として「職場の上司・管理者」が最も高い割合**。
→ ✗ 選択肢4「相談したい内容：正社員は適性・適職、正社員以外は将来のCPラン」は誤り。〔GS5〕

**問12** ✗ 「新入社員がOFF-JT最多」が不適切かどうか判断できず選択肢1を選択
→ ◯ 令和5年度能力開発基本調査：能力開発・人材育成に関する問題点の最多は「**人材を育成しても辞めてしまう**」。
→ ✗ 選択肢3「技術革新や業務変更が頻繁なため人材育成が無駄になる」が不適切（最多ではない）が正答。〔GS6〕

**問16** ✗ 女性労働力率のM字カーブを「U字」と誤解
→ ◯ 日本女性の年齢階級別労働力率の特徴は**M字カーブ**（育児期に低下し再び上昇）。UでもLでもない。〔GS7〕

**問17** ✗ 「日本の65歳以上男性労働力率は韓国より高いが欧米より低い」を正答と誤選択
→ ◯ 正答：「65歳以上男性の労働力率を1985年と2022年で比較すると、アメリカ・カナダ・イギリス・ドイツは**大幅に上昇**しているが、**日本ではわずかに低くなっている**」。〔GS8〕

**問18** ✗ 「女性の非正規雇用労働者数は2015年以降減少傾向」を正答と誤選択
→ ◯ 正答：「2023年における女性の非労働力人口のうち、**就業希望者数は完全失業者数よりも多い**」。
→ ✗ 女性の非正規雇用労働者数は2015年以降も**増加**傾向（減少ではない）。〔GS9〕

**問19** ✗ 「労働力人口比率＝全人口に占める労働力人口の割合」を正答と誤選択
→ ◯ 正答：**完全失業率＝労働力人口に占める完全失業者の割合**。
→ ✗ 労働力人口比率の分母は「全人口」ではなく「**15歳以上人口**」。〔GS10〕

### キャリア理論

**問4** ✗ 「自我同一性」が社会認知的キャリア理論（SCCT）に含まれると誤解し「学習経験」を選択
→ ◯ SCCTの主要概念：**学習経験・自己効力感・結果期待**。
→ ✗ **自我同一性（アイデンティティ）はエリクソン**の概念であり、SCCTには含まれない。〔GT2〕

**問6** ✗ ホランドとYG性格検査の関係を誤り選択肢3を選択
→ ◯ 正答：「職務満足・安定性・業績は、**個人のパーソナリティとその人の働く環境との一致度**に依拠すると考えた」（選択肢4）。
→ ✗ **YG性格検査（矢田部ギルフォード）はホランドとは無関係**。ホランドはVPI・SDSを開発。〔GT3〕

**問31** ✗ シャイン「3サイクル」の説明を正答と誤選択
→ ◯ 正答：シュロスバーグは「転機のプロセスには、**始まりと終わりの間にニュートラルゾーンがあり、個人がどの位置にいるかを見極めることが重要**」とした。
→ ✗ 選択肢1（シャインの3サイクル）は正しい記述だが「最も適切」ではない。〔GT4〕

### カウンセリング

**問8** ✗ バーン（Berne, E.）のキーワードを「自己一致」と誤解
→ ◯ 正答：**フロイト（Freud, S.）・・・リビドー**。
→ ✗ **自己一致はロジャーズ（Rogers, C.R.）**のキーワード。バーン（Berne, E.）は**交流分析（TA）**。〔GC8〕

**問9** ✗ 「行動療法の基本的な説明」が不適切に見えず選択肢1を選択
→ ◯ 正答（不適切なもの）：「観察学習は、モデルを観察し、**観察者自身もその行動を実施したところ報酬が得られることによって成立する**」→ 誤り。
→ ◯ バンデューラの観察学習（モデリング）：**モデルを観察するだけで学習が成立する**（実際に行動・報酬を得なくてもよい）。〔GC9〕

**問34** ✗ 「意味の反映」を積極技法と誤解
→ ◯ マイクロカウンセリング「**積極技法**」：フィードバック・直面化・自己開示・指示・情報提供・解釈など。
→ ✗ **意味の反映は傾聴技法（基本的傾聴連鎖）**に属する。積極技法ではない。〔GC10〕
