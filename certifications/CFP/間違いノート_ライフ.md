## ⓪ 計算問題「掛け忘れ」3大事故 ★★★最優先（第2回で連続失点）

- **給付率の掛け忘れ**：✗賃金日額×日数で止める → ◯**出生時育休/育休=×67%**・介護休業=×67%。給付額は必ず**給付率まで掛ける**。(2025-2 Q20)
- **付加年金も繰上げ/繰下げ減額の対象**：✗付加年金を満額のまま加算 → ◯老齢基礎と**同率(繰上0.4%/月・繰下0.7%/月)で増減**。付加年金200円×納付月数にも掛ける。(2025-2 Q35)
- **賃金日額の算定基礎＝分母は常に÷180**：✗**賃金総額を出勤日数(140日等)で割る**(2周目の実ミス)・賞与退職金を算入 → ◯**÷180(固定値・6ヶ月×30)**。**賃金支払基礎日数11日未満の月は除外**してさかのぼる・**賞与/退職金は不算入**。手順「①賞与退職金除外 ②11日未満月除外 ③÷180」。**180は"暦日数"ではない**(暦日数で割るのは労災の給付基礎日額)。出勤日数で割るとWが過大→上限額8,635に張り付き誤答。(2025-2 Q19)
- **国保の介護分は第2号(40-64歳)のみ**：世帯員のうち40-64歳だけ所得割・均等割を計上。未就学児は均等割5割軽減。(2025-2 Q26)
- **小規模共済の増額時=階層計算**：✗総額で一括 → ◯**増額部分はその時点からの納付月数**で口数別に共済金額を算出し合算。(2025-2 Q44)
- **老齢給付の計上漏れに注意**：老齢給付＝老齢基礎＋報酬比例＋**経過的加算**(＋条件を満たせば加給/振替加算)。✗基礎+報酬比例で止める → ◯**経過的加算**(定額部分相当1,734×月数 − 831,700×20-60歳の厚年月数/480)を必ず足す。**振替加算は夫が65歳到達後**に発生(妻が年上なら妻65歳時点ではまだ付かない)。(模試4 Q36)
- **20歳前/60歳以後の厚年期間**は老齢"基礎"年金には反映されない(報酬比例・経過的加算には反映)。基礎の月数計算で除外。(模試4 Q36)

## ⓪-2 2025-1 2周目で残った穴（継続失点★／後退）

**継続失点★（1周目・2周目とも誤答＝最優先）**
- **奨学金 減額返還**：✗通算10年/120ヶ月 → ◯**通算15年/180ヶ月**。(2025-1 Q12)
- **一般教育訓練給付**：✗修了後の追加給付あり → ◯**追加給付なし**(20%・上限10万のみ)。追加給付は専門実践(最大70/80%)。(2025-1 Q15)
- **法定休日**：✗週2回/4週8日 → ◯**毎週1回または4週4日**。割増は35%以上。(2025-1 Q17)
- **任意継続**：✗傷病・出産手当金が出る/未納で翌月1日喪失 → ◯**手当金は原則出ない**(継続給付除く)／**未納は納付期日の翌日に喪失**。(2025-1 Q29)
- **資格喪失後の給付は本人ベース**：✗被扶養者だった人の出産に家族出産育児一時金 → ◯**出ない**(喪失後は被扶養者の概念なし)。埋葬料と同じ構造。(2025-1 Q32)

- **DB(確定給付企業年金)脱退一時金**：✗3年超の加入期間を要件に → ◯**「3年以下」の加入期間まで**。老齢給付金は20年超を要件にできない。(2025-1 Q42)★DC/DB弱点
- **間接法 営業CF**：✗符号を逆に → ◯**資産(売掛金・棚卸)増加=CFマイナス／負債(買掛)減少=CFマイナス**。減価償却費はプラス加算。(2025-1 Q46)
- **遺言書保管制度の閲覧**：✗相続放棄者は不可 → ◯**放棄者も関係相続人等として閲覧可**。本人出頭(代理不可)・無封。(2025-1 Q48)
- **特養(介護老人福祉施設)**：✗要介護2以上 → ◯**原則要介護3以上**。老健・介護医療院は要介護1以上。(2025-1 Q49)

## ① 健康保険・介護保険 ★最頻出の穴

- **被扶養者**：✗130万円ちょうどOK → ◯**130万円未満**(60歳以上/障害者は180万円未満)かつ被保険者年収の1/2未満。配偶者子孫兄弟姉妹直系尊属は**別居でも可**(仕送り要件)、それ以外は同居必須。失業給付・年金も収入にカウント。
- **任意継続**：◯**継続2ヶ月以上**加入＋**20日以内**申請＋**最長2年**＋**全額自己負担**。この3数字をセットで。
  - ★**任意継続の標準報酬月額**：✗喪失前1年間の平均 → ◯**①喪失時の標準報酬月額**と**②前年9/30の全被保険者の平均**の**いずれか低い方**。(模試4 Q25)
- **傷病手当金**：✗支給開始から1年6ヶ月 → ◯**通算1年6ヶ月**(2022改正)。日額**2/3**、連続3日待期後4日目から。
- **出産育児一時金**：◯**50万円**(2023/4〜)。
- **埋葬料と埋葬費**：✗どちらも一律5万円 → ◯**埋葬料(生計維持者あり)=一律5万円／埋葬費(生計維持者なし)=実費・上限5万円**。(模試4 Q31)
- **訪問看護療養費**：「厚労省令で定める者」に**医師は含まれない**(看護師・保健師・助産師・理学療法士等)。(模試4 Q31)
- **国年第1号の産前産後免除**：✗全額免除期間 → ◯**保険料納付済期間扱い**(老齢基礎年金に満額反映)。出産月の前月から4ヶ月(多胎は3ヶ月前から6ヶ月)。(模試4 Q27)
- **介護保険**：✗第2号もいつでも給付 → ◯**第2号(40-64歳)は特定疾病(16種)が原因のときだけ**。利用者負担1〜3割、ケアプラン作成は自己負担なし。保険者は市町村。
- **高額療養費の世帯合算（計算）**：✗70歳未満本人の医療費だけで限度額算定 → ◯限度額式 80,100+(**総医療費**-267,000)×1% の**総医療費は合算対象の世帯全体**(70歳以上分も含む)。手順は①70-74外来(個人)→②70歳以上世帯→③70歳未満(21,000円以上)を加えた世帯全体、の順に算定。(模試4 Q34)

## ② 雇用保険・労働保険 ★

- **基本手当 受給要件**：一般=**2年で12ヶ月**／特定受給資格者・特定理由離職者=**1年で6ヶ月**。
- **算定基礎期間**：✗ブランクも通算 → ◯**空白1年超で前後は通算されない**。
- **所定給付日数**：一般(自己都合)=10年未満90/10-20年120/20年〜150日。特定受給資格者は年齢×期間で細分(60-65歳20年〜=240日)。
- **給付制限**：自己都合は原則**2ヶ月**(5年内2回まで)。会社都合なし。待期7日。
- **一般教育訓練給付**：✗30%/上限20万 → ◯**20%・上限10万円**。離職後**1年以内**受講開始。
- **高年齢雇用継続**：賃金が60歳時の**75%未満**で支給。最大率は**2025/4から10%**(従来15%)。
- **介護休業給付**：◯**67%**・通算**93日**・**3回**まで分割可。
- **労災**：休業補償は**4日目から給付基礎日額の60%**(+特別支給金20%=実質80%)。保険料**全額事業主負担**、パートも対象。
- **労基**：割増=時間外25%/法定休日35%/深夜25%/月60h超50%。法定労働時間1日8h週40h。最低賃金=月給÷所定労働時間で比較。

## ③ 公的年金 ★

- **繰上げ/繰下げ**：✗繰上0.5% → ◯繰上**0.4%/月**(2022〜)・繰下**0.7%/月**(最大75歳84%増)。受給資格期間**10年**。
- **在職定時改定**：✗9月分から改定 → ◯基準日**9/1**・改定は**10月分(12月支払)から**(2022導入)。
- **在職老齢年金**：基本月額＋総報酬月額相当額が**支給停止調整額**超で停止。停止額=(基本月額+総報酬月額相当額−調整額)÷2。★**調整額は改正で変動**：2024=50万→2025=51万→**2026=65万**(大幅引上げ)。受験年度の額を必ず確認。古い過去問は出題時点の額(50/51万)が正解。
- **iDeCo拠出限度**：第1号68,000円(国年基金等と合算)/第2号企業年金なし23,000/DB等あり20,000/第3号23,000/公務員20,000(2024/12〜)。原則60歳から受給(加入10年未満は後ろ倒し)。
- ★**確定拠出年金の受取課税**(第2回・模試4で連続失点の継続弱点)：✗一時金=一時所得 → ◯**一時金=退職所得**(退職所得控除)／**年金=雑所得**(公的年金等控除)。企業型のマッチング拠出とiDeCoは**選択制で併用不可**。資格喪失後**6ヶ月以内**に手続きしないと国民年金基金連合会へ自動移換。(2025-2 Q42・模試4 Q42)
- **個人年金(生保)の必要経費**：受取年金×(払込総額÷総支給見込額)。公的年金等は雑所得・控除速算表は65歳前後で別。

## ④ 成年後見・遺言 ★

- **任意後見の解除**：✗いつでも家裁許可必要 → ◯**監督人選任“前”は公証人認証の書面で自由(許可不要)**、**選任“後”は家裁許可**。任意後見人報酬は特約なければ無報酬。任意後見は**監督人選任時から効力発生**。
- **自筆証書遺言**：本文は自書、**財産目録はPC可/通帳コピー可**(各頁署名押印)。**法務局保管制度を使えば検認不要**。公正証書遺言も検認不要・証人2人。

## ⑤ 係数・キャッシュフロー表

- **6係数**：毎年いくら積立?=**減債基金係数**／毎年いくら受取?=**資本回収係数**(ローン返済額)。目標額に必要な現在元本=現価係数(一括)/年金現価係数(取崩原資)。
- **CF表**：将来値=現在値×(1+**変動率**)ⁿ。**貯蓄残高=前年残高×(1+運用率)±当年収支**。可処分所得=年収−(所得税+住民税+社会保険料)※生保料は引かない。
