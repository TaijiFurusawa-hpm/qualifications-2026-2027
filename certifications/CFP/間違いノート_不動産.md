## ① 税制：取得・保有（数値が命）

- **不動産取得税**：✗相続も贈与も同じ → ◯**相続は非課税・贈与は課税**。税率は土地/住宅3%・住宅以外4%。
- **取得税の宅地課税標準**：✗税率を半分 → ◯**課税標準×1/2**（税率は3%のまま）。
- **固定資産税の住宅用地**：✗一律1/6 → ◯**200㎡以下=1/6、200㎡超部分=1/3**。
- **固定資産税の納税義務者**：◯**1/1時点の登記簿上の所有者**。評価替えは3年に1度。
- **登録免許税・抵当権設定**：✗0.4%のまま → ◯住宅特例で**0.1%**。土地売買移転は本則2%→軽減**1.5%**(2029/3/31まで)。
- **印紙税・贈与契約書**：◆**"何の贈与か"で分かれる**→**金銭の贈与契約書=不課税(印紙不要)／不動産の贈与契約書=課税(記載金額なし1通200円)**。建物賃貸借契約書は非課税/土地賃貸借は課税。不動産売買契約書は軽減あり/**住宅ローン(金銭消費貸借契約書)は軽減なし(本則)**＝2025-2 Q37肢2が誤り。
- ◆**住宅取得等資金贈与の非課税限度(2024〜2026)**：**省エネ等住宅=1,000万円／それ以外=500万円**。設例に「省エネ等住宅に該当しない」とあれば500万(=5,000千円)＝2025-2 Q36。但書を読んでも金額を覚えていないと落とす→「省エネ1,000/一般500」をセット暗記。
- **不動産所得の経費**：✗所得税住民税も経費 → ◯**所得税住民税は不可・事業税は可**。附属設備の償却は**定額法のみ**。**売るための立退料は譲渡費用(経費不可)**。

## ② 税制：譲渡の特例（要件・併用・繰延/控除の別）

- **譲渡所得の基本**：取得費不明→**概算取得費=収入×5%**。相続取得は**被相続人の取得日・取得費を引継ぐ**。所有期間は**譲渡年1/1**で判定（5年超で長期20%）。
- ★**3,000万円控除**：✗所有期間10年必要 → ◯**所有期間要件なし**。「住まなくなって3年後の年末まで」譲渡OK。除外は配偶者・直系血族・**生計一**親族→**生計を一にしない弟への譲渡は適用可**。家屋とともになら敷地一部の区分譲渡もOK(2025-1 Q37)。
- ★**固定資産の交換特例(所法58条)**：交換差金は「**いずれか高い方の価額の20%以内**」(乙土地の20%でない)。差金を"受け取った"側のみ差金部分課税。**取得資産に取得費・取得日が引継がれる**。相手のその後の売却は適用に無関係(2025-1 Q40)。
- **軽減税率(10年超)**：◯3,000万控除後、**6,000万以下=14%**。3,000万控除と**併用可**。
- **買換え特例**：✗非課税 → ◯**課税繰延べ**。買換資産の床面積**50㎡以上**(40㎡不可)。**合計所得5,000万円以下**(R6改正)。**住宅ローン控除と併用不可**。
- **空き家特例(3,000万)**：✗新耐震も対象/賃貸でもOK → ◯**1981/5/31以前建築**が対象・**相続後に貸付(無償でも)・居住・事業に供したら不可**。譲渡期限は相続から3年後の年末。
- **立体買換え特例**：✗80%非課税/店舗のみOK → ◯**床面積1/2以上が住宅**必要・所有期間問わず・全額充当で**課税繰延べ**。
- **併用横断**：3,000万控除＋軽減税率=◯／3,000万控除＋住宅ローン控除=✗／買換え＋住宅ローン控除=✗。
- ★**定期借地の保証金**：自宅(非事業)に充当→経済的利益を**不動産所得に算入**。金融資産で運用→計算不要。✗利子税の利率 → ◯適正利率は**基準年利率(長期国債利回り基準)**で利子税の利率ではない(2025-1 Q44・2周連続失点／資料訂正済)。
- **事業用資産の買換え特例(措法37条)**：✗80%非課税 → ◯**繰延割合80%(20%は課税)**。土地は譲渡面積の**5倍まで**対象。計算: 課税収入=譲渡額−買換額×80%／対応取得費=（取得費+譲渡費用)×(課税収入÷譲渡額)／課税所得=差引。買換資産の取得価額は**引継ぎ計算**(将来の譲渡益が膨らむ)。売買契約後に死亡でも相続人が期限内取得・事業供用で適用可。

## ③ 民法・宅建業法 ★第2回で失点増(5→8問)・最重点

- **媒介契約**：✗一般も3ヶ月上限 → ◯**一般は期間制限なし**。専任=登録7日/報告2週に1回、専属専任=登録5日/報告1週に1回。
- **手付金**：売主業者は代金**2割まで**。原則解約手付。売主解除は**手付倍額の現実の提供**が必要(口頭不可)。
- **契約不適合責任**：✗特約は無効 → ◯**責任を負わない特約も有効**(知って告げなかった事実には及ばない)。引渡し前の滅失は買主が代金支払拒否可。
- **共有**：✗持分処分も全員同意 → ◯**自己の持分は単独で処分可**。軽微変更=過半数、変更/処分=全員同意。10年の資材置場賃貸借=全員同意。
- **相隣・枝と根**：✗枝も根も自由に切れる → ◯**枝は原則催告必要(例外3つ)・根は承諾不要で切れる**。設備設置権(2023改正)あり。
- **抵当権**：✗利息は常に2年分 → ◯**「他に債権者がいる場合」最後の2年分**。物上代位(火災保険金)は**払渡し前の差押え**が要件。借地権(賃借権)には抵当権設定不可。
- ◆**請負契約(建物建築)**：✗建物は解除できない → ◯**改正民法で建物も解除可**。不適合は知った時から1年以内通知。完成前は注文者がいつでも損賠して解除可。
- ◆**連帯保証**：◯催告の抗弁権・検索の抗弁権・分別の利益が**ない**(普通保証との違い)。個人根保証は**極度額の定め必須**(なければ無効)。
- ◆**定期建物賃貸借(定借)**：✗口頭でOK → ◯**書面契約＋事前に書面交付して説明**(怠ると普通借家に)。更新なし。普通借家と違い**1年未満も有効**。200㎡未満居住用は転勤等で借主中途解約可。
- ★**賃貸人の地位の移転(民605条の2)**：賃貸中物件の譲渡に**賃借人の承諾は不要**。賃借権は**引渡しで対抗可**(登記不要)。新賃貸人が**賃料請求するには所有権移転登記が必要**(通知では足りない)。**敷金返還債務は特約なく当然に新賃貸人へ承継**＝2025-2 Q22肢4。
- ★**法定更新後の期間(借26条)**：建物賃貸借が法定更新されると更新後は**「期間の定めのないもの」**(従前の2年でない)＝2025-2 Q23肢1の誤り。造作買取請求権の排除特約は有効。必要費は事前同意なく直ちに償還請求可(民608条)。
- ★**区分所有・占有者(賃借人)の権利義務**：占有者は使用方法につき区分所有者と**同一の義務**(締結前後で区別なし)。**管理費の支払義務なし**。共同利益背反行為の停止請求は**占有者単独では不可**＝2025-2 Q24肢3。集会で意見陳述できるのは**利害関係がある場合のみ**(管理費増額は利害関係なし)。
- ◆**都市計画道路予定地**：✗全く建てられない → ◯**2階建て以下・地階なし・容易に除却できる建物(木造・鉄骨造等)なら知事許可で建築可**。**基準適合でも許可は必ず必要**(許可不要が誤り)。**RC造・地階ありは許可を受けても不可**＝2025-2 Q32(正解は地階なし1階・鉄骨造)。

## ④ 有効活用 ◆第2回で失点

- ◆**建設協力金方式**：テナントから建設資金を借りて建物を建て賃貸。土地=**貸家建付地**、建物=**貸家**で相続評価(相続対策)。減価償却費は**法定耐用年数**で計算(賃貸借期間でない)。中途解約で返還不要となった額は所得。
- ◆**事業用定期借地権(借地借家法23条)**：✗住宅もOK → ◯**事業用に限る(居住用不可)**。長期型(23条1項)=30年以上50年未満、短期型(23条2項)=10年以上30年未満。**必ず公正証書**で契約。
- **事業受託方式**：デベロッパーが立案・資金斡旋・設計施工・運営管理を行う。土地信託方式は固定資産税の納税義務者が**受託者(信託銀行)**。

## ⑤ 登記

- ★**登記事項証明書/図面**：✗証明書はオンラインで電磁的記録で届く → ◯**書面で郵送/窓口交付**。ただし**地積測量図等の図面はオンラインで交付請求できる**(2025-1 Q19)。要約書には認証文・職印なし。登記に公信力なし。
- ★**分筆登記**：✗1人で申請可/抵当権抹消が必要 → ◯**共有地は全員で申請**・抵当権付きでも分筆可。**分筆では登記識別情報は通知されない**(表示の登記)。分筆だけでは単独名義にならず持分移転登記が別途必要(2025-1 Q21)。
- **借地権**：✗更新料の支払義務あり → ◯**更新料の法的義務なし**。建物を借地権者名義で登記すれば借地権を対抗できる。地代増額請求時は相当と認める額を払えば足りる。
- ◆**所有権移転登記**：登記識別情報は**買主に通知**(売主の申出でも登記済証は交付されない)。住所変更があれば前提として名義人住所変更登記が必要。
- ★**筆界特定制度**：筆界は所有者の合意では変更できない(公法上の境界＝2025-2 Q27肢1の誤り)。**関係人＝隣接地の所有権登記名義人等で、対象土地の抵当権者は当然には含まれない**(Q27肢2は不適切。⚠️旧版「抵当権者も関係人」を訂正)。筆界特定書の写しは**利害関係なくても誰でも交付請求可**。確定した境界確定訴訟の判決と抵触する範囲で筆界特定は失効(Q27肢4＝正解)。

## ⑥ 価格評定・建築基準法 ★計算で継続失点

- ◆**収益還元法・還元利回りと収益価格は逆相関**：直接還元法は**収益価格＝純収益÷還元利回り**。**還元利回りが上昇すると収益価格は下落**(2025-2 Q12肢4「上昇で価格上昇」が誤り＝正解)。修繕費計上・空室増→純収益減→価格下落／空室率を低く見積・**保証金等の預り金の運用益増→純収益増→価格上昇**(肢3は正しい記述なので選ばない)。
- ◆**取引事例比較法/積算法の計算**：時点修正・標準化補正(間口狭小×不整形地)・地域要因・個別的要因の補正率を順に乗じる。複合計算の手順を固める(第1回Q13・第2回Q14,15で失点)。
- ◆**容積率の計算**：前面道路幅員×法定乗数と指定容積率の小さい方。複数用途地域は加重平均。セットバック・角地緩和に注意。
- ◆**開発許可**：市街化区域は1000㎡以上で許可必要だが**区画形質の変更を伴わなければ不要**。市街化調整区域は規模に関わらず原則必要(300㎡未満でも)。
- ★**土地区画整理法**：✗1人では施行できない → ◯**個人施行は1人でもできる**。仮換地指定後は従前地を使用収益不可でも第三者へ売却は可(2025-1 Q30)。
- ★**公的価格/固定資産課税台帳**：✗定期借地権は借地権割合を乗じて評価 → ◯**定期借地権は残存期間に応じた複利現価方式**(借地権割合方式でない)。借家人は課税台帳で**建物＋その敷地**を閲覧可。評価替えは固定資産税のみ3年に1度(2025-1 Q14)。
- ◆**地価公示(令和7年・時事)**：全国平均・三大都市圏とも全用途/住宅地/商業地が**4年連続上昇・上昇幅拡大**。**大阪圏も上昇幅拡大**(縮小でない)(2025-1 Q50・後退)。
