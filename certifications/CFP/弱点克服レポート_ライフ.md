## 【弱点0】計算問題の「掛け忘れ」事故 ★★★最優先（第2回で連続失点・最も得点直結）

スコア28/50(ボーダーちょうど)。失点22問中、計算5問(Q19,20,26,35,44)。**いずれも"あと一手"の掛け忘れ・場合分け漏れ**で、知識ではなく手順の問題。ここを潰すのが最短の得点アップ。

| 事故 | ✗やりがち | ◯正しい手順 | 出題例 |
|---|---|---|---|
| 給付率の掛け忘れ | 賃金日額×日数で止める | 出生時育休/育休/介護休業は**×67%**まで掛ける | 2025-2 Q20(→234,500円) |
| 付加年金の減額漏れ | 付加年金を満額のまま加算 | 付加年金も**繰上0.4%/繰下0.7%で同率増減**(×0.76等) | 2025-2 Q35(→615,233円) |
| 賃金日額の算定基礎 | 6ヶ月そのまま÷180・賞与算入 | **11日未満の月を除外**してさかのぼる・**賞与/退職金は不算入** | 2025-2 Q19(→6,381円) |
| 国保介護分の対象者 | 世帯全員で介護分計上 | **介護分は第2号(40-64歳)のみ**・未就学児は均等割5割軽減 | 2025-2 Q26(→1,009,200円) |
| 小規模共済の増額 | 総納付月数で一括計算 | **増額分はその時点からの納付月数**で口数別に算出し合算(階層方式) | 2025-2 Q44(→11,067,900円) |

**チェックリスト（計算問題を解いたら必ず確認）**
1. 給付額に**給付率(%)**を掛けたか？
2. 年金で繰上げ/繰下げがある問題は、**付加年金・加給等にも増減率**を反映したか？
3. 賃金日額は**11日未満の月を除外**し、**賞与/退職金を除外**したか？
4. 「世帯/被保険者ごと」の**対象者の場合分け**(第2号のみ等)を正しく切ったか？

## 【弱点1】社会保険：健康保険 ★最重点（失点6問・最多）

### 1-1. 被扶養者の要件

| 項目 | 内容 |
|---|---|
| 収入要件 | 年収 **130万円未満**（60歳以上または障害者は **180万円未満**）、かつ被保険者の年収の**1/2未満**（別居の場合は仕送り額未満） |
| 同一世帯不要の親族 | 配偶者・子・孫・兄弟姉妹・直系尊属 → **同居でなくても**被扶養者になれる |
| 同一世帯が必要な親族 | 上記以外（甥姪・配偶者の父母等）→ **同居が条件** |
| 内縁関係 | 内縁の配偶者も被扶養者になれる |

**ひっかけ要点**
- 「130万円未満」は**未満**（130万円ちょうどはダメ）。60歳以上/障害者は180万円未満。
- 配偶者・子・孫・兄弟姉妹・直系尊属は**別居でも可**（仕送り要件あり）。それ以外は同居必須。
- 収入には**雇用保険の失業給付・公的年金・健康保険の傷病手当金も含む**（非課税でもカウント）。

**【出題例】** 2025-1 Q28（協会けんぽの被扶養者要件）

## 【弱点3】公的年金 ★最重点（失点5問）

### 3-2. 在職老齢年金・在職定時改定 ★（持続ミスで誤答）

|---|---|---|
| 2024年度 | **50万円** | |
| 2025年度 | **51万円** | 賃金・物価スライドで小幅改定 |
| 2026年度 | **65万円** | ★大幅引上げ（在職老齢年金の見直し）。働く高齢者の年金が停止されにくくなる |

**改正の覚え方・狙い**
- 支給停止調整額は毎年度改定される（賃金変動に連動）。**受験する年度の最新額**を必ず確認すること。
- 2026年度の**50万→51万→65万**の流れは、「高齢者就労を促すため、年金カットの基準を引き上げる」政策方向。65万円超で初めて支給停止が始まる。
- 計算式は不変：**支給停止額＝(基本月額＋総報酬月額相当額−支給停止調整額)÷2**。変わるのは「支給停止調整額」の数値だけ。
- 過去問は出題時点の額で作られているため、古い過去問では50万・51万が正解。**改正で数値が動く論点**と認識し、年度を必ず紐づけて記憶する。

**【出題例】** 弱点模試 ライフ41（在職定時改定・老齢厚生年金額の改定）

## 【弱点4】法律：成年後見・遺言 ★（持続ミスでも登場）

### 4-1. 成年後見制度

| 種類 | 内容 |
|---|---|
| 法定後見 | 判断能力の程度で**後見・保佐・補助**の3類型。家庭裁判所が選任 |
| 任意後見 | 本人が判断能力のあるうちに**公正証書**で契約。**任意後見監督人が選任された時**から効力発生 |

**任意後見契約の解除のタイミング（持続ミスで誤答）**
- **監督人選任“前”**：本人・任意後見受任者は、**公証人の認証を受けた書面**でいつでも解除可（家裁の許可不要）。
- **監督人選任“後”**：**家庭裁判所の許可**を得て解除。
- 任意後見人の報酬：契約で定める。**特約がなければ無報酬**（請求できない）。

**ひっかけ要点**
- 監督人選任の**前後で解除手続きが違う**（前=認証書面で自由、後=家裁許可）。
- 任意後見は監督人が選任されて初めて効力が生じる。

**【出題例】** 2025-1 Q47（任意後見人の選任・成年後見）／弱点模試 ライフ47（成年後見・任意後見契約）

## 【弱点5】係数計算・キャッシュフロー表 ★（持続ミスでも計算失点）

### 5-1. 6つの係数

| 係数 | 用途 |
|---|---|
| 終価係数 | 現在の額→将来の額（複利運用後） |
| 現価係数 | 目標額に必要な**現在の元本** |
| 年金終価係数 | 毎年積立→将来の合計額 |
| 減債基金係数 | 目標額のための**毎年の積立額** |
| 年金現価係数 | 毎年取崩す原資として必要な**現在の元本** |
| 資本回収係数 | 元本を一定期間で取り崩す**毎年の受取額**（ローン返済額） |

**ひっかけ要点**
- 「毎年いくら積み立てる？」＝減債基金係数／「毎年いくら受け取れる？」＝資本回収係数。
- 住宅ローンの毎年返済額＝借入額×資本回収係数。残債＝毎月返済額×年金現価係数。

### 5-2. キャッシュフロー表

- 将来の収入・支出＝現在値×(1+**変動率**)ⁿ
- 年間収支＝可処分所得−支出
- **貯蓄残高＝前年残高×(1+運用率)±当年収支**
- 可処分所得＝年収−(所得税＋住民税＋社会保険料) ※生命保険料等は引かない

**ひっかけ要点**
- 貯蓄残高は「前年残高に運用率をかけてから収支を加減」。
- 可処分所得から引くのは**税金と社会保険料のみ**。

**【出題例】** 2025-1 Q14（CF表の作成）／Q17（変更後CF表・係数計算）/Q46（CF表計算）／弱点模試 ライフ6（CF表計算）・ライフ9（住宅ローン繰上）

## 【弱点6】第2回で露呈した新規・横断論点（要件・数値の単純暗記で取り切る）

### 6-1. 社会保険の適用・給付（細則）
- **適用事業所**：✗美容業8人で強制適用 → ◯**美容業・飲食業等のサービス業は非適用業種**で強制適用にならない。**士業(税理士等)は5人以上で強制適用**(2022/10〜)。任意適用は従業員1/2以上の同意＋厚労大臣認可。(2025-2 Q29)
- **資格喪失後の継続給付**：✗喪失後の被扶養者死亡に家族埋葬料 → ◯**喪失後は被扶養者がいないため家族埋葬料は出ない**。喪失後の埋葬料は本人(被保険者だった者)死亡時のみ。出産手当金受給者が退職日に出勤すると継続給付不可。(2025-2 Q32)
- **労働安全衛生法**：✗80h超の面接指導は希望の有無を問わず → ◯**労働者本人の申出**が要件。深夜業健診は配置替え時＋6ヶ月以内ごと、定期健診は1年以内ごと。(2025-2 Q24)
- **介護休業の対象家族**：配偶者・父母・子・祖父母・兄弟姉妹・孫・配偶者の父母。**「配偶者の祖父母」は含まない**。(2025-2 Q25)

### 6-2. 公的年金・企業年金（制度）
- **ねんきん定期便/ネット**：見込額に**加給年金は反映されない**(正)。封書は節目年齢**35・45・59歳**(40/50/59ではない)。(2025-2 Q41)
- **企業型DC**：✗マッチング拠出とiDeCo併用可 → ◯**マッチング拠出を選択している場合iDeCo併用不可(選択制)**。受給開始上限は**75歳**。掛金は月単位のほか**賞与時期のみの拠出も可**。(2025-2 Q42)
- **小規模企業共済**：個人事業を子へ全部譲渡＝共済金A。掛金増額時は**口数×納付月数別の共済金額**を合算(階層計算)。(2025-2 Q44)

### 6-3. 高齢者の住まい・後見・事業資金（法律）
- **サービス付き高齢者向け住宅(サ高住)**：入居要件は**60歳以上**または60歳未満の要介護/要支援認定者(「65歳以上」は誤り)。床面積は共用充実で18㎡以上に緩和。(2025-2 Q49)
- **終身建物賃貸借**：賃借人は**60歳以上**、公正証書等で締結、賃借人死亡で終了。解約は事業者から**知事等の承認**が必要、賃借人の解約申入れは**1ヶ月**以上前。(2025-2 Q50)
- **任意後見・財産管理委任**：✗任意後見受任者は死亡届を出せない → ◯**任意後見受任者も死亡届の届出人になれる**(2020改正)。財産管理委任契約は公正証書は義務でない。(2025-2 Q48)
- **信用保証協会**：✗信用保証で団信加入が必須 → ◯**団信加入は義務でない**。農業が主たる事業は保証対象外。担保で保全十分なら代表者連帯保証を不要にできる。(2025-2 Q46)

### 6-4. 個人情報・労働・教育資金（その他）
- **個人情報保護法**：利用目的の変更は「関連性を有すると合理的に認められる範囲」内のみ。**範囲を超える変更は本人同意があってもできない**(新規取得扱い)。(2025-2 Q2)
- **障害者雇用**：端数は**切り捨て**、重度障害者は1人を2人にカウント、障害者雇用納付金は月額**5万円**(調整金29,000円と混同しない)。(2025-2 Q3)
- **教育一般貸付(日本政策金融公庫)**：扶養する子が2人以内で自宅外通学(予定)者がいると**世帯年収(所得)上限が緩和**。自宅外通学の融資限度は450万円。(2025-2 Q14)
- **最低賃金法**：派遣労働者は**派遣先**の事業場所在地の地域別/特定最低賃金が適用(派遣元ではない)。(2025-2 Q17)
