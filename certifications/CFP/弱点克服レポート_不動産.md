## 【弱点1】不動産の取得・保有に係る税制 ★最重点（33%×4回停滞）

### 1-1. 不動産取得税

| 項目 | 内容 |
|---|---|
| 課税主体 | 都道府県（地方税） |
| 課税対象 | 不動産の取得（売買・**贈与**・交換・新築・増改築） |
| **非課税の取得** | **相続**による取得／法人の合併／包括遺贈 など |
| 税率（本則） | 4% |
| 税率（特例） | 土地・住宅 **3%**（2027/3/31まで）／住宅以外の家屋は4% |
| 課税標準 | 固定資産税評価額 |
| 宅地の課税標準特例 | 宅地は課税標準が **価格×1/2**（2027/3/31まで） |
| 住宅の課税標準控除 | 新築住宅は1戸につき **1,200万円控除**（認定長期優良住宅は1,300万円） |

**ひっかけ要点**
- 「相続は非課税／贈与は課税」… ここが最頻出。贈与は不動産取得税かかる。
- 税率3%は「土地と住宅」。**店舗・事務所など住宅以外の家屋は4%**。
- 宅地の1/2は「課税標準」の話。税率を半分にするのではない。

**【出題例】** 2025-1 Q32（土地建物の取得税率＝いずれも3%は誤り。建物が住宅なら3%だが、論点は「宅地課税標準1/2」「新築建物の課税標準1/3」等の組合せ）／弱点模試 不動産36（贈与でも取得税は課税）

---

### 1-2. 固定資産税

| 項目 | 内容 |
|---|---|
| 課税主体 | 市町村（東京23区は都） |
| 納税義務者 | **1月1日（賦課期日）時点の登記簿上の所有者** |
| 税率（標準） | 1.4% |
| 課税標準 | 固定資産税評価額（3年に1度評価替え） |
| **住宅用地の課税標準特例** | 小規模住宅用地（200㎡以下の部分）＝価格×**1/6**／一般住宅用地（200㎡超の部分）＝価格×**1/3** |
| 新築住宅の税額減額 | 一定要件で居住部分120㎡まで、3年間（マンション等は5年間）税額1/2 |

**ひっかけ要点**
- 「1/6」は小規模住宅用地（200㎡以下）、「1/3」は200㎡超部分。取得税の宅地1/2と混同しない。
- 課税標準特例（1/6, 1/3）と新築減額（税額1/2）は別物。前者は課税標準、後者は税額。
- 評価替えは**3年に1度**（地価公示・路線価は毎年）。

**【出題例】** 2025-1 Q32（マンション敷地の固定資産税課税標準1/6 → 「6分の1」が正しい記述として登場）

### 1-3. 登録免許税

| 登記の種類 | 本則税率 | 住宅特例（自己居住用） |
|---|---|---|
| 所有権保存（新築） | 0.4% | 0.15% |
| 所有権移転（売買・土地） | 2.0%→**1.5%**（土地売買の軽減、**2029/3/31まで**※令和6年度改正で延長） | — |
| 所有権移転（売買・建物） | 2.0% | 0.3%（住宅特例） |
| 所有権移転（相続） | 0.4% | — |
| 所有権移転（贈与） | 2.0% | — |
| 抵当権設定 | 0.4% | **0.1%**（住宅特例） |

- 課税標準：所有権登記は固定資産税評価額／抵当権設定は債権金額。
- 住宅用家屋の軽減は、床面積50㎡以上・新築または取得後1年以内の登記等の要件。

**ひっかけ要点**
- 抵当権設定の本則は0.4%、住宅特例で0.1%。
- 土地売買の移転は本則2.0%だが軽減で1.5%。
- 相続による移転は0.4%、贈与は2.0%（取得税の非課税/課税とセットで整理）。

**【出題例】** 2025-1 Q31（登録免許税・印紙税の組合せ。土地所有権移転1.5%軽減・抵当権設定0.1%軽減が論点）

### 1-4. 印紙税

| 文書 | 課税/非課税 |
|---|---|
| 不動産売買契約書 | 課税（記載金額に応じ階段税率、軽減措置あり） |
| 不動産の**贈与契約書** | **課税**（記載金額のない契約書として **1通200円**） |
| 金銭消費貸借契約書（住宅ローン） | 課税（軽減措置なし＝本則） |
| 建物賃貸借契約書 | **非課税**（土地の賃貸借は課税） |
| 委任状・媒介契約書 | 非課税 |

**ひっかけ要点**
- **贈与契約書は「記載金額なし」扱いで200円課税**。非課税ではない。← 持続ミスで誤答
- 建物の賃貸借契約書は非課税。土地の賃貸借（借地権設定）は課税。
- 印紙の貼付がなくても契約は有効。ただし過怠税（本来の印紙税額＋その2倍＝3倍）。消印なしは印紙額と同額の過怠税。

**【出題例】** 弱点模試 不動産37（贈与契約書は印紙税課税・200円）／2025-1 Q31

### 1-5. 不動産所得（賃貸経営）の必要経費 ★過去問で失点

不動産賃貸の所得税で「必要経費になる/ならない」の判定。

| 項目 | 必要経費 |
|---|---|
| 固定資産税・都市計画税・不動産取得税・登録免許税・印紙税 | **算入できる** |
| その不動産所得に係る所得税・住民税 | 算入**できない** |
| 事業税 | 算入できる |
| 借入金の利子（元本は不可） | 利子のみ算入可 |
| 減価償却費 | 算入できる |
| 青色事業専従者給与（要件該当時） | 算入できる |
| **立退料** | 賃貸継続のため／建替えのための立退料は経費算入可。**譲渡のための立退料は譲渡費用（経費不可）** |

**ひっかけ要点**
- 「不動産所得に係る所得税・住民税は経費不可、事業税は可」。
- 立退料は「目的」で判定。賃貸継続・建替え＝経費／**売るための立退き＝譲渡費用**。
- 賃貸用建物の附属設備の減価償却は、2016/4以降取得分は**定額法のみ**（定率法の選択不可）。

**【出題例】** 2025-1 Q9（ア:取得税等は経費・所得税住民税は不可＝適切／イ:附属設備は定額法のみ→定率法選択可は誤り／エ:立退料は譲渡なら経費不可）

## 【弱点2】不動産の譲渡に係る税制・各種特例 ★最重点（譲渡税制50%下落）

### 2-0. 譲渡所得の基本

- 譲渡所得 ＝ 譲渡収入 −（**取得費** ＋ **譲渡費用**）− 特別控除
- **概算取得費**：取得費が不明なときは **収入金額×5%** を取得費にできる。
- **取得日・取得費の引継ぎ**：相続・贈与で取得した資産は、**被相続人・贈与者の取得日と取得費を引き継ぐ**（単純承認の相続も同様）。

| 区分 | 所有期間（譲渡年の1/1時点） | 税率（所得税+住民税、復興税別） |
|---|---|---|
| 短期譲渡 | 5年以下 | 39%（所得税30%＋住民税9%） |
| 長期譲渡 | 5年超 | 20%（所得税15%＋住民税5%） |

**ひっかけ要点**
- 所有期間は「譲渡した年の1月1日現在」で判定（実際の譲渡日ではない）。
- 相続取得の場合、被相続人の取得時期を引き継ぐ＝長期になりやすい。

**【出題例】** 2025-1 Q36（相続取得で取得費不明→概算取得費5%。譲渡価額56,000千円・取得費5%・譲渡費用2,200千円・3,000万円控除後の課税譲渡所得を計算）

---

### 2-1. 居住用財産を譲渡した場合の3,000万円特別控除

| 項目 | 内容 |
|---|---|
| 控除額 | 譲渡所得から最高 **3,000万円** |
| 所有期間要件 | **なし**（短期所有でも可） |
| 居住要件 | 現に居住／住まなくなった日から **3年を経過する日の属する年の12/31** までの譲渡 |
| 適用除外（親族間） | **配偶者・直系血族・生計一親族等への譲渡は不可**（兄弟姉妹で生計別なら適用可の場合あり） |
| 併用 | 「軽減税率の特例」とは**併用可**。「住宅ローン控除」とは**併用不可** |
| 連年適用制限 | 前年・前々年にこの特例等を受けていると不可（3年に1度） |

**ひっかけ要点**
- 所有期間の要件はない（軽減税率の特例には10年超要件がある）。
- 「住まなくなって3年後の年末まで」… 期限の言い回しが頻出。
- 家屋所有者と敷地所有者が異なる場合、原則は家屋所有者が控除。一定要件（生計一・同居等）で敷地所有者にも控除枠を配分。

**【出題例】** 弱点模試 不動産41（居住用財産の特例・所有者が異なる場合の要件）

---

### 2-2. 居住用財産の軽減税率の特例（10年超所有）

- 所有期間が譲渡年1/1で **10年超**の居住用財産。
- 3,000万円控除後の課税長期譲渡所得につき、
  - **6,000万円以下の部分：14%**（所得税10%＋住民税4%）
  - 6,000万円超の部分：20%
- 3,000万円特別控除と**併用可能**。

---

### 2-3. 特定の居住用財産の買換え特例

| 項目 | 内容 |
|---|---|
| 内容 | 譲渡益への課税を**繰り延べ**（非課税ではない・課税の先送り） |
| 譲渡資産の要件 | 所有期間10年超・居住期間10年以上・譲渡対価1億円以下 |
| 買換資産の要件 | 床面積 **50㎡以上**（建物）・土地500㎡以下・一定期日までに取得し居住 |
| 合計所得金額 | **5,000万円以下**（令和6年度改正で追加された要件） |
| 居住開始期限 | 取得した年の翌年12/31まで |
| 併用 | **住宅ローン控除とは併用不可**／3,000万円控除・軽減税率とも**併用不可** |

**ひっかけ要点**
- 「繰延べ」であって「非課税」ではない。将来売却時に繰り延べた分が課税される。
- 買換資産の床面積要件は**50㎡以上**（40㎡では不可）。
- この特例を使うと**住宅ローン控除は使えない**。
- 合計所得金額**5,000万円以下**（令和6年度改正で追加）。5,000万円超は不可。

**【出題例】** 2025-1 Q35（買換え特例。床面積40㎡では不可／合計所得2,000万円なら適用可（5,000万円以下のため）／居住2027年では遅い／住宅ローン控除併用不可）
※当時の正解肢は「合計所得2,000万円でも適用可（肢1が誤り＝正解）」。現行は5,000万円以下要件があるが2,000万円は範囲内のため結論は不変。

### 2-4. 被相続人の居住用財産（空き家）に係る譲渡所得の特例

| 項目 | 内容 |
|---|---|
| 控除額 | 最高 **3,000万円**（相続人が3人以上なら1人2,000万円） |
| 対象家屋 | **1981年（昭和56年）5月31日以前**建築（旧耐震）・区分所有建物でない |
| 要件 | 相続開始直前に被相続人が**1人で居住**（老人ホーム入所も一定要件で可） |
| 譲渡時の状態 | **耐震改修して譲渡**、または**家屋を取壊して更地で譲渡** |
| 譲渡期限 | 相続開始日から **3年を経過する日の属する年の12/31** まで |
| 適用除外 | 相続〜譲渡までに**事業・貸付・居住の用に供したら不可**（無償の貸付でも不可） |
| 譲渡対価 | 1億円以下 |

**ひっかけ要点**
- 旧耐震（1981/5/31以前）が対象。新耐震は対象外。
- **相続後に貸し付けたら（無償でも）アウト**。← 持続ミスで誤答した論点。
- 「軽減税率の特例」とは併用不可だが、空き家特例は**併用に関する独自ルール**あり（同一年で居住用3,000万控除と合わせて最大6,000万円まで等）。

**【出題例】** 2025-1 Q37（空き家特例の不適切な記述。取得費引継ぎ・区分譲渡・親族間譲渡・一時的賃貸など）／弱点模試 不動産40

### 2-5. 立体買換え（等価交換）の特例 — 租税特別措置法37条の5

| 項目 | 内容 |
|---|---|
| 内容 | 既成市街地等内で土地等を譲渡し、地上の中高層耐火建築物を取得 → 譲渡益の課税繰延べ |
| 買換資産の住宅要件 | 建築物の**床面積の1/2以上が住宅**であること |
| 譲渡資産の所有期間 | **問わない**（10年以下でも可） |
| 全額買換時 | 譲渡金額の全部を買換資産取得に充てれば**課税繰延べ（譲渡がなかったものとされる）**。「80%だけ非課税」ではない |

**ひっかけ要点**
- 床面積の1/2以上が住宅でなければならない（店舗のみ取得は不可）。
- 所有期間10年以下でも適用可（譲渡所得の長短とは別の制度）。
- 「繰延べ」であり「80%非課税」ではない。

**【出題例】** 2025-1 Q48（立体買換え特例。床面積1/2以上住宅／所有期間問わず／全額充当で課税繰延べ）

### 2-6. 特例併用の整理（横断暗記）

| 組合せ | 可否 |
|---|---|
| 3,000万円控除 ＋ 軽減税率（10年超） | **併用可** |
| 3,000万円控除 ＋ 住宅ローン控除 | **併用不可** |
| 買換え特例 ＋ 住宅ローン控除 | **併用不可** |

### 2-7. 定期借地権の保証金の経済的利益（所得税）

- 借地権設定時に受け取る**保証金（返還を要する金銭）**は、運用方法により所得税の扱いが変わる。
- 自宅建築費等の**非事業用**に充てた場合：保証金の経済的利益を**不動産所得の収入金額に算入**。
- 預貯金・公社債など**金融資産で運用**した場合：経済的利益の計算は**不要**。
- 事業用資産の取得に充てた場合：収入算入＋同額を必要経費算入（相殺）。
- 適用利率は**「基準年利率」（長期国債の利回りを基にした国税庁の定める利率）**による。**利子税の利率ではない**。

> ⚠️ **訂正（2026-06-12）**：旧版で「適用利率は利子税の利率による」としていたが**誤り**。2025-1 Q44の肢4「適正な利率は利子税の計算と同じ年利率による」は**不適切＝正解肢**であり、保証金の経済的利益の適正利率は利子税の利率とは別物（基準年利率）。

**【出題例】** 2025-1 Q44（保証金の経済的利益の所得税課税。肢4「利子税の利率による」が不適切＝正解）

---

### 2-8. 特定の事業用資産の買換え特例（措法37条）★2025-2で3問集中・計算で失点

居住用の買換え特例とは別制度。**事業用**の土地建物を譲渡し、一定の事業用資産に買い換えた場合、譲渡益の一部の課税を**繰り延べる**（非課税ではない）。

| 項目 | 内容 |
|---|---|
| 課税繰延割合 | 原則**80%**（一定の地域間移転は70%・90%等）。買換えた部分の譲渡益の80%が繰延、**20%は課税** |
| 譲渡資産・買換資産 | ともに**事業用**であること（自用・継続的な貸付け等）。棚卸資産は不可 |
| 買換資産の取得期限 | 譲渡年の前年〜翌年中に取得し、取得日から1年以内に事業供用 |
| 土地の面積制限 | 買換える土地は、譲渡した土地面積の**5倍まで**が対象（超える部分は対象外） |
| 所有期間 | 譲渡資産は譲渡年1/1で**所有期間5年超**（一部の号で要件あり） |
| 他特例との関係 | 3,000万円控除等とは選択（併用不可） |

#### 計算手順（★Q38・39で落とした計算）

設例（2025-2 Q38）：譲渡価額200,000／取得費40,000／譲渡費用8,000／買換資産購入額150,000／繰延割合80%

**① 収入金額とみなす額** ＝ 譲渡価額 − (買換資産取得額 × 繰延割合)
　※譲渡額 > 買換額 のとき。買換額が譲渡額以上なら「譲渡価額 ×(1−繰延割合)」
- 買換150,000 ≦ 譲渡200,000 なので：
- 課税対象収入 ＝ 200,000 − (150,000×80%) ＝ 200,000 − 120,000 ＝ **80,000**

**② 対応する取得費・譲渡費用** ＝ (取得費＋譲渡費用) × (課税対象収入 ÷ 譲渡価額)
- (40,000＋8,000) × (80,000 ÷ 200,000) ＝ 48,000 × 0.4 ＝ **19,200**

**③ 課税譲渡所得** ＝ ① − ② ＝ 80,000 − 19,200 ＝ **60,800千円**（Q38正解=3）

#### 買換資産が引き継ぐ取得価額（Q39の計算）

買換資産の取得価額 ＝ ①買換えに充てた譲渡資産の引継取得費 ＋ ②買換資産の実際の取得額のうち課税された部分
- 簡便には：買換資産取得額 − 繰り延べられた譲渡益
- Q39正解=3（58,800千円）。「実際の取得額150,000」をそのまま引き継ぐのではなく、繰延分だけ取得費が圧縮される＝**将来売却時に繰延益が顕在化**する仕組み。

**ひっかけ要点**
- **繰延割合80%は「非課税」ではない**。20%は課税され、残りは買換資産の取得費圧縮で将来に繰り越す。
- 土地は**譲渡面積の5倍まで**。超過分は特例対象外。
- 買換資産の取得価額は実額ではなく**引き継ぎ計算**（だから将来の譲渡益が大きくなる）。
- 売買契約後に本人が死亡しても、**相続人が期限内に取得・事業供用すれば適用可**（Q40の論点）。

**【出題例】** 2025-2 Q38（譲渡所得の計算）／Q39（引継取得価額の計算）／Q40（適用要件の正誤）

## 【弱点3】民法・宅地建物取引業法（売買・媒介・相隣・共有）

### 3-1. 宅建業法：媒介契約

| 種類 | 有効期間 | 指定流通機構登録 | 業務報告 | 自己発見取引 |
|---|---|---|---|---|
| 一般媒介 | **制限なし**（標準約款は3ヶ月） | 義務なし | 義務なし | 可 |
| 専任媒介 | **3ヶ月以内** | 契約日から**7日以内**（休業日除く） | **2週間に1回以上** | 可 |
| 専属専任媒介 | 3ヶ月以内 | 契約日から**5日以内**（休業日除く） | **1週間に1回以上** | **不可（業者経由のみ）** |

**ひっかけ要点**
- 一般媒介の有効期間に法的上限はない（専任・専属専任は3ヶ月）。
- 指定流通機構登録：専任7日・専属専任5日（休業日含まず）。
- 報告頻度：専任は2週に1回、専属専任は1週に1回。

**【出題例】** 2025-1 Q18（媒介契約の適切な記述。一般媒介の有効期間3ヶ月超は3ヶ月になる→誤り 等）／弱点模試 不動産21

### 3-2. 宅建業法：手付金

- 売主が宅建業者の場合、手付額は**代金の2割（10分の2）まで**。
- 手付は原則**解約手付**と解される。
- 解約手付による解除：相手方が履行に着手するまで、買主は**手付放棄**、売主は**手付倍返し**で解除可。
- 売主からの解除は、**現実に手付倍額を提供**する必要（口頭の意思表示では不可）。
- 売主・買主双方とも宅建業者でない取引には、上記2割制限等の宅建業法規制は適用されない。

**【出題例】** 2025-1 Q16（手付金。倍額の現実の提供が必要 等）

### 3-3. 民法：売買・契約不適合責任

- 売主は、引き渡した目的物が種類・品質・数量に関して契約内容に不適合の場合、**追完責任・代金減額・損害賠償・解除**を負う。
- 買主の権利行使：不適合を知った時から**1年以内に通知**（種類・品質）。
- **契約不適合責任を負わない旨の特約も有効**（ただし売主が知りながら告げなかった事実には及ばない／宅建業者が売主の場合は制限）。
- 危険負担：引渡し後に当事者双方の責めなく滅失 → 買主は代金支払を拒めない。引渡し前の滅失 → 買主は代金支払を拒める／解除可。

**【出題例】** 2025-1 Q17（売買契約の不適合責任・危険負担。引渡し前後で危険の移転が変わる）

### 3-4. 民法：共有

| 行為 | 必要な同意 |
|---|---|
| 保存行為（修繕等） | 各共有者が単独で可 |
| 管理行為（軽微変更含む・短期賃貸借等） | **持分価格の過半数** |
| 変更行為（軽微以外）・処分 | **共有者全員の同意** |
| 自己の持分の処分（売却等） | **単独で可**（他の共有者の同意不要） |

- 共有物分割：協議が調わなければ裁判所に請求。**現物分割・賠償分割・換価分割**。
- 賃貸借：短期（土地5年以下・建物3年以下等）は管理行為（過半数）。長期（10年の賃貸借等）は変更行為に近く全員同意が必要な場合あり。

**ひっかけ要点**
- 自己の持分は他の共有者の同意なく自由に売却できる。
- 「軽微な変更」は過半数でよい（2023年民法改正）。
- 建物所有目的でない資材置場として10年の賃貸借 → 全員同意が必要。

**【出題例】** 2025-1 Q20（共有の不適切な記述。持分処分は単独可／軽微変更は過半数 等）

### 3-5. 民法：相隣関係（2023年改正反映）

- **越境した竹木の枝**：原則は所有者に切除を催告。ただし①催告後相当期間内に切除しない、②所有者不明・所在不明、③急迫の事情、のいずれかで**自ら切り取れる**。
- **越境した竹木の根**：**承諾なく自ら切り取れる**（昔から根は自由）。
- **隣地使用権**：境界付近の建築・修繕等のため必要な範囲で隣地使用を請求可。損害が生じれば償金。
- **ライフライン設備設置権**：他の土地に設備を設置しなければ電気・ガス・水道等の継続的給付を受けられない場合、必要な範囲で他の土地に設備設置可（2023年改正）。

**ひっかけ要点**
- **枝は原則催告必要（例外あり）／根は承諾不要で切れる**。この枝・根の差が頻出。
- 設備設置権は2023年改正の新論点。

**【出題例】** 2025-1 Q22（相隣関係の不適切な記述。枝と根の切除ルール／隣地使用／設備設置）

### 3-6. 民法・借地借家法：抵当権

- 抵当権の効力は、原則として抵当不動産に**付加して一体となった物（付合物）に及ぶ**。
- 被担保債権の利息：他に債権者がいる場合、**満期となった最後の2年分**に制限（他に債権者がいなければ制限なし）。
- 物上代位：抵当目的物の滅失で生じる火災保険金請求権等に物上代位できるが、**払渡し前に差押え**が必要。
- 抵当権は債権を担保する物権。借地権（賃借権）には抵当権を設定できない（地上権・永小作権には設定可）。

**ひっかけ要点**
- 利息の「最後の2年分」制限は**他に後順位抵当権者等がいる場合**の話。いなければ制限されない。
- 火災保険金への物上代位は**差押えが要件**。

**【出題例】** 2025-1 Q24（抵当権の適切な記述。利息2年分制限／物上代位の差押え／付加一体物）

### 3-7. 民法：請負契約（建物の建築） ★2025-2で失点

- 注文住宅の建築は**請負契約**。請負人は仕事を完成させる義務、注文者は報酬支払義務。
- **契約不適合責任**：請負人が種類・品質に関し契約不適合の仕事をした場合、注文者は**追完請求・報酬減額・損害賠償・解除**ができる。
- 期間制限：注文者が不適合を**知った時から1年以内に通知**（引渡し時に請負人が知っていた等は除く）。
- **注文者の任意解除**：請負人が仕事を完成しない間は、注文者はいつでも損害を賠償して契約を解除できる。
- 建物その他土地工作物でも、不適合を理由とする解除は可能（2017年改正前は不可だったが改正で可能に）。

**ひっかけ要点**
- 旧法では「建物は契約解除できない」だったが、**改正民法では建物も解除可**。
- 不適合の通知は「知った時から1年以内」。

**【出題例】** 2025-2 Q21（注文建物に関する請負人の法的責任）

### 3-8. 民法：連帯保証 ★2025-2で失点

### 3-9. 借地借家法：定期建物賃貸借（定期借家） ★2025-2で失点

| 項目 | 普通借家 | 定期借家 |

| 事前説明 | 不要 | 契約前に「更新がなく期間満了で終了する」旨を**書面交付して説明**（怠ると定借として無効＝普通借家に） |

**ひっかけ要点**

**ひっかけ要点**

## 【弱点4】不動産登記

### 4-1. 登記事項証明書・登記の基礎

- 登記事項証明書は誰でも交付請求できる（手数料納付）。オンライン請求も可だが、**証明書自体は書面で郵送または窓口交付**（オンラインで電磁的記録として送信されるわけではない）。
- 登記事項要約書：登記記録の概要。認証文・登記官の職印は**付されない**（証明書とは異なる）。
- 地図（14条地図）・地積測量図：オンラインで請求情報を提供して交付請求可。
- 公信力：日本の不動産登記に**公信力はない**（対抗力はある）。

**ひっかけ要点**
- 登記事項証明書はオンライン請求しても「書面」で受け取る。
- 要約書には認証文・職印がない。

**【出題例】** 2025-1 Q19（登記事項証明書等の適切な記述）

---

### 4-2. 分筆の登記

- 分筆登記は**表示に関する登記**で、**表題部所有者または所有権登記名義人が申請**。共有地なら**共有者全員で申請**（単独申請は不可）。
- 抵当権が設定された土地でも分筆は可能（抵当権抹消は不要。分筆後は各筆に抵当権が及ぶ）。
- 分筆登記が完了すると、新たに生じた土地に**登記識別情報は通知されない**（分筆は持分移転を伴わないため。ただし設問により扱い注意）。
- 分筆だけでは持分は変わらない。共有を解消して単独所有にするには**持分移転登記が別途必要**。

**ひっかけ要点**
- 共有地の分筆は**全員で申請**（1人では不可）。
- 分筆＋持分移転をしないと単独名義にはできない。

**【出題例】** 2025-1 Q21（分筆登記の適切な記述）／弱点模試 不動産28（所有権移転登記の名義・識別情報）

---

### 4-3. 借地権・借地借家法（登記がらみ）

- 借地権の対抗要件：土地に借地権の登記がなくても、**借地上の建物を借地権者名義で登記**すれば第三者に対抗できる。
- 法定更新：存続期間満了時に借地権者が使用継続し、地主が遅滞なく異議を述べなければ更新（建物がある場合）。更新後の期間は最初20年・以降10年。
- 更新料：**法律上の支払義務はない**（特約や慣習があれば別）。
- 地代増減請求：協議が調わないとき、増額請求された借地人は**相当と認める額を支払えば足りる**（裁判確定まで）。

**ひっかけ要点**
- 更新料の支払義務は法律にはない。← 頻出ひっかけ。
- 建物の登記があれば借地権を対抗できる（借地権自体の登記は不要）。

**【出題例】** 2025-1 Q23（借地権の不適切な記述。更新料の法的義務 等）／弱点模試 不動産20

## 【追補】2025-2 不動産 2周目で露呈した論点（2026-06-13）

2周目で**継続失点12問・新規失点3問**。テーマ別の核を以下に集約（既存の弱点1〜4と重複する論点は該当節を参照）。

### A. 鑑定評価：還元利回りと収益価格は逆相関 ★Q12（基本式・新規失点）

直接還元法：**収益価格 ＝ 純収益 ÷ 還元利回り**
- 還元利回りが**上昇**すると分母が大きくなり、収益価格は**下落**する（Q12肢4「上昇する」が誤り＝正解）。
- 純収益を増やす要因（空室率を低く見積る・保証金/預り金の運用益増）は収益価格を**上昇**させる。
- 修繕費の計上・空室増は純収益を減らし収益価格を**下落**させる。
- ※保証金等の預り金の増加は「運用益が純収益を押し上げる」ので価格上昇（Q12肢3は正しい記述＝選んではいけない）。

### B. 賃貸人の地位の移転 ★Q22（新規失点）

賃貸中の不動産が譲渡され賃貸人の地位が移転する場合（民605条の2）：

| 論点 | 結論 |
|---|---|
| 賃借人の承諾 | **不要**（地位は当然に移転） |
| 賃借権の対抗 | 建物の**引渡し**があれば対抗可（賃借権登記は不要・借31条） |
| 新賃貸人が賃料請求するための要件 | **所有権移転登記が必要**（賃借人への通知では足りない・民605条の2第3項） |
| 敷金返還債務 | 承継特約がなくても**当然に新賃貸人へ承継**（民605条の2第4項）＝Q22肢4＝正解 |

### C. 借地借家法：法定更新後の期間 ★Q23（継続失点）

- 建物賃貸借が**法定更新**された場合、更新後は**「期間の定めのないもの」**となる（借26条1項）。従前の2年が引き継がれるのではない（Q23肢1が誤り）。
- 造作買取請求権（借33条）を排除する特約は**有効**（任意規定・強行規定でない）。
- 通常損耗・経年変化の原状回復義務は賃借人にない（民621条）。明確な特約がなければ負わない。
- 賃借人が支出した**必要費は事前同意なく直ちに償還請求できる**（民608条1項）＝Q23肢4＝正解。

### D. 区分所有法：占有者（賃借人）の権利義務 ★Q24（継続失点）

- 占有者は、建物等の使用方法につき**区分所有者が規約・集会決議に基づき負う義務と同一の義務**を負う（区46条2項。締結前後で区別しない＝肢1の誤り）。
- 占有者は**管理費の支払義務を負わない**（区分所有者の債務＝肢2の誤り）。
- 共同利益背反行為の停止請求（区57条）は集会決議に基づくもので、**占有者が単独で請求はできない**（肢3＝正解）。
- 占有者が集会で意見を述べられるのは**会議の目的につき利害関係を有する場合**のみ（区44条）。管理費（区分所有者負担）増額は占有者に利害関係がなく意見陳述権なし（肢4の誤り）。

### E. 筆界特定制度 ★Q27（新規失点・★既存資料の誤りを訂正）

- **筆界は公法上の境界で、所有者の合意では変更できない**（合意で動かせるのは所有権界＝Q27肢1の誤り）。
- 筆界特定の**「関係人」＝関係土地（隣接地）の所有権登記名義人等**。**対象土地の抵当権者は関係人に当然には含まれない**（Q27肢2は不適切とされた）。
- 筆界特定書（手続記録）の写しは、**利害関係がなくても誰でも交付請求できる**（肢3の誤り）。
- 筆界特定後に境界確定訴訟の判決が確定すると、**抵触する範囲で筆界特定は効力を失う**（不登148条＝Q27肢4＝正解）。

> ⚠️ **訂正（2026-06-13）**：間違いノートの「抵当権者も関係人として意見・資料提出可」は誤り。2025-2 Q27で対象土地の抵当権者を関係人とする肢が不適切と判定された。関係人は隣接地の所有権登記名義人等に限られる。

### F. 住宅取得等資金贈与の非課税限度額 ★Q36（継続失点・数値暗記）

直系尊属から住宅取得等資金の贈与を受けた場合の非課税限度（2024〜2026年）：

| 区分 | 非課税限度額 |
|---|---|
| **省エネ等住宅** | **1,000万円** |
| 上記以外の住宅 | **500万円** |

- Q36は設例に「省エネ等住宅に該当しない」と明記 → **500万円＝5,000千円（正解1）**。
- 但書は読んでいたが金額を記憶しておらず1,000万を選択＝**数値暗記の精度が課題**。「省エネ1,000／一般500」をセットで固定する。

### G. 事業用買換え特例の計算 → 弱点2-8（既出）を参照

Q38（譲渡所得60,800千円）・Q39（引継取得価額58,800千円）は【弱点2-8】で計算手順を詳述済み。2周連続失点のため**最優先で計算反復**。

### H. 印紙税：課税文書の区分 ★Q37（迷い・正答）

- **金銭の贈与契約書 → 不課税（印紙不要）**。
- **不動産の贈与契約書 → 課税（記載金額のない第1号文書＝1通200円）**。
- 不動産売買契約書 → 軽減税率あり。**住宅ローン（金銭消費貸借契約書）→ 軽減特例なし（本則）**＝Q37肢2が誤り＝正解。
- Q37は正答したが「不動産贈与契約書(課税)」と「金銭贈与契約書(不課税)」を混同して迷った。**贈与は"何の贈与か"で印紙の要否が分かれる**点を固定。

### その他の継続失点（既存節を参照）

- Q3 品確法：指定住宅紛争処理機関を使えるのは**「建設」住宅性能評価書**交付住宅（「設計」評価書では不可）。
- Q16 公的価格：(ア)倍率地域の借地権割合は**相続税路線価図のA〜G**／(イ)宅建業者の価額意見の根拠は公示価格等に限定されない＝両方不適切（→弱点4参照）。
- Q17 広告規制：**建築確認前は広告自体不可**。建蔽率・容積率は必須表示事項でなく表示なしでも可（正肢3）。
- Q21 請負：引渡し前に**双方無責の地震で全損・完成不能なら報酬請求できない**（民634条・利益を受ける可分部分なし）＝肢2が不適切＝正解（→弱点3-7参照）。
- Q42 事業用定期借地権：**専ら事業用・居住用を除く**。社員寮（居住用）は床面積1/2以上事業用でも不可（肢2が誤り）（→弱点3-11・間違いノート④参照）。
- Q50 令和7年版土地白書（統計）：肢4（主要都市の店舗賃料）が不適切。暗記型で白書要点整理が必要。
