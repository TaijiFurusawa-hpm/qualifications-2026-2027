## 【弱点1】個人事業税 ★最重点（3回連続0%）

| 項目 | 内容 |
|---|---|
| 課税主体 | 都道府県（地方税・賦課課税） |
| 課税対象 | 法定70業種の事業所得・（事業的規模の）不動産所得 |
| **課税対象外** | 給与所得・譲渡所得・一時所得・林業・一部の農業 等 |
| 課税標準 | （事業所得＋不動産所得）−損失の繰越控除−事業主控除 |
| **事業主控除** | **年290万円**（年途中の開業・廃業は月割） |
| **青色申告特別控除** | **適用しない（足し戻す）** ← 所得税との最大の違い |
| 専従者給与 | 控除する（青色・白色とも） |
| 税率 | 第一種5%・第二種4%・第三種5%（一部3%） |

**ひっかけ要点**
- 所得税の青色申告特別控除（最大65万）は**事業税では引かない**。ここが毎回の失点源。
- 事業所得が赤字でも不動産所得が黒字なら、同じ事業税対象内で**損益通算**して課税標準を出す。
- 計算の型: `(事業＋不動産)−繰越損失−290万 ×税率`。

**【計算例】** 事業▲20万＋不動産355万−繰越30万−290万＝15万 → ×5%＝**7,500円**。
**【出題例】** 模試 問37（細川さん・第一種5%）。

## 【弱点3】損益通算 ◆悪化中（100→50→0%）

**通算できる損失は「不・事・山・譲」のみ**（富士山上）。配当・一時・雑・給与の損失は不可。

| 論点 | 内容 |
|---|---|
| 不動産所得の損失 | **土地等取得の負債利子部分は通算不可**（建物分はOK） |
| 生活に通常必要でない資産 | 別荘・ゴルフ会員権・1個30万円超の書画骨董等の譲渡損は**通算不可** |
| 生活用動産 | 30万円以下の家具・衣類・車両等の譲渡損益は**非課税**（通算以前） |
| 通算の順序 | ①経常所得グループ内 ②グループ間（総合長期譲渡・一時は1/2前で通算） ③山林・退職 |

**【計算例】** 事業210＋不動産▲320＋一時340＋生活用車両譲渡▲20。<br>車両損は不可。不動産▲320を事業210と通算→残▲110。一時(1/2前340)と2次通算→230。総所得＝230×1/2＝**115万円**。
**【出題例】** 模試 問30（総所得金額）、問31（損益通算後の所得税額・雑所得の損失は切捨て）。

## 【弱点4】法人税 ★配点大（17→50→67%）

### 4-1. 租税公課の損金算入/不算入

| 損金不算入 | 損金算入 |
|---|---|
| 法人税・地方法人税・法人住民税の本税 | 法人事業税・特別法人事業税 |
| 延滞税・加算税・延滞金・過怠税 | 固定資産税・不動産取得税・自動車税 |
| 罰金・科料・過料・交通反則金（業務関連） | 印紙税（本税）・事業所税 |

### 4-2. 交際費（中小法人）
- **800万円**と**接待飲食費の50%**の**有利な方**を損金算入。
- 1人1万円以下の飲食費は交際費から除外（社内接待は除く）。

### 4-3. 減価償却（少額減価償却資産の特例）
- 中小・青色は**取得価額30万円未満・年間合計300万円**まで全額損金。
- 30万円「未満」なので**300千円ちょうどは不可**。中古資産は簡便法耐用年数(法定−経過)+経過×20%。

### 4-4. 役員給与
- 損金算入は**定期同額・事前確定届出・業績連動**のみ。改定は期首から3ヶ月以内。
- 期中の減額（業績悪化改定事由でない）は、減額後を基準に**超過部分が損金不算入**。

### 4-5. 貸倒損失
- 法律上（民事再生の認可で切捨て）＝損金／形式上（書面で債務免除通知）＝損金。
- 事実上（全額回収不能）は**担保物を処分した後**でないと不可。

### 4-6. 繰越欠損金
- **青色欠損のみ10年繰越**（2018/4以後開始）。白色の年の欠損は対象外。中小は所得全額まで控除可。

**【出題例】** 模試 問41（租税公課5,310千円）、問42（交際費）、問43（役員給与・交通反則金）、問44（少額減価償却545,000円）、問45（貸倒）、問46（繰越欠損金1,900千円）。

## 【弱点6】株式等の譲渡所得と配当所得 ★配点大（33→67→67%）

| 論点 | 内容 |
|---|---|
| 口座区分と申告 | 源泉あり特定口座も**他口座と通算するには確定申告が必要** |
| 譲渡損失の繰越 | **3年**。申告分離を選択した上場配当・特定公社債利子と通算（古い損失から先に） |
| 配当の申告（上場3%未満） | 申告不要／総合／申告分離を選択 |
| 配当の申告（非上場・少額） | 1回≦10万円×月数/12 で申告不要可 |
| 大口株主（3%以上） | **総合課税のみ**（申告不要・申告分離不可） |
| NISA内の損失 | **ないものとされ通算・繰越不可** |
| 申告分離 vs 総合 | 分離=損益通算可・配当控除不可／総合=配当控除可・損益通算不可 |

**【出題例】** 模試 問23（口座区分）、問24（譲渡損失の3年繰越）、問25（配当の申告不要判定）。

## 【弱点11】不動産所得・事業所得 ★（不67→83→50%／事80→60→80%）

- **事業的規模（5棟10室）**で資産損失全額経費・青色専従者給与・青特65万・回収不能家賃を当年経費が可能。

## Tier2（重要）— リタイアメント・財務諸表

### リタイアメント
- **退職所得**＝(収入−退職所得控除)×1/2。控除＝20年以下40万×年、20年超800万+70万×(年−20)。勤続**1年未満切上げ・休職含む**。
- **複数退職金（会社+小規模共済等）同年受給**: 最も長い勤続期間で控除、重複しない期間を加算。
- **一時払養老等を5年以内解約**: 金融類似商品→20.315%源泉分離。5年超は一時所得。
- **公的年金等控除**: 65歳以上330万以下=110万（65歳未満130万以下=60万）。
- **死亡保険金**: 負担者=被保険者・受取人=相続人→相続税／契約者=受取人・被保険者別→所得税(一時)／三者別→贈与税。

**【出題例】** 問20（公的年金＋一時所得226万）、問21（保険の課税関係）、問22（複数退職金465万）。

### 財務諸表の読みこなし
- 損益分岐点売上高＝固定費÷限界利益率（限界利益率=1−変動費率）。
- 流動比率=流動資産÷流動負債／固定比率=固定資産÷自己資本／固定長期適合率=固定資産÷(自己資本+固定負債)／自己資本比率=自己資本÷総資本。
- **利益増≠CF好転**、売上減でも原価抑制で総利益率は上昇しうる。

**【出題例】** 問49（固定比率133.3%が正・75%は誤り）、問50（売上高販管費率の低下が適切）。

### 追加2. 修正申告・更正・加算税・延滞税【問36】

| 論点 | 正しい取扱い |
|---|---|
| 修正申告の期限 | 更正の処分があるまで・**徴収権の時効（原則5年）まで**提出可（「3年で不可」は誤り） |

| 調査通知後〜更正予知前の修正申告 | 過少申告加算税5%（一定額超10%）に軽減 |
| 更正の予知後・更正処分 | 過少申告加算税10%(超過分15%)／隠蔽仮装なら**重加算税35%** |
| 延滞税の計算期間 | 法定納期限の翌日から**完納の日まで**（「修正申告書提出日まで」と限定するのは誤り。※期限内申告分は1年経過後の特例控除期間あり） |

**ひっかけ要点**
- 「調査通知前の自主修正」は**加算税ゼロ**（早期是正のインセンティブ）。ここを「課税対象」と書く肢は誤り。
- **更正処分でも加算税の対象**になる（修正申告しなければ免れる、ではない）。

### 追加3. 法人契約の生命保険の「経済的利益」判定【問48】

> **大原則：役員・従業員に経済的利益（給与）が生じるのは、保険金受取人が「被保険者またはその遺族」の場合のみ。受取人が法人なら経済的利益はゼロ。**

| 保険 | 契約者 | 被保険者 | 受取人 | 経済的利益（給与課税） |
|---|---|---|---|---|
| A | 法人 | 全従業員 | **法人** | **0**（法人受取＝利益は法人へ。山岸は役員で対象外でもある） |
| B | 法人 | 全役員全従業員 | **法人** | **0**（法人受取） |
| C | 法人 | 山岸 | **遺族(法定相続人)** | **保険料10万全額が給与**（遺族受取＋**役員のみ対象＝普遍的加入でない→福利厚生費にならない**） |

→ 山岸の経済的利益＝**10万円のみ**（A・Bは法人受取で対象外）。

**福利厚生費 vs 給与の分岐**
- 遺族受取の定期保険でも、**全員加入（普遍的加入）なら福利厚生費**で給与課税なし。
- **役員のみ・特定者のみ**を対象とすると、その分は**給与（経済的利益）**として課税。

### 追加4. 製造原価報告書・損益計算書・財務指標【問49・50】

**空欄補充の手順（計算問題＝得点源）**
