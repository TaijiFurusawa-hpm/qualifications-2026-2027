## 📚 CC理論・概念集（語彙）

### 🤝 関係構築
- **ラポール** = 信頼関係・心理的安全性の醸成
  - 構成手法: **共感／ペーシング／ミラーリング／傾聴**
- **受容・共感・寄り添い** … CC の基本態度
- **コンプリメント** … 単に労う／褒めるではない
  - **事実ベースで** プロセス・姿勢・**内面（責任感・気遣い等）** を **クリーンランゲージ** で具体的に返す
  - 例: 職探し行動だけでなく「母親への気遣い・責任感」にコンプリメント

### 🔄 認知・行動アプローチ
- **リフレーミング** = ネガティブ事象を別視点で再定義
  - 例: 派遣終了 → **転機** ／ 何にもなれない → **何にでもなれる可能性**
- **イラショナルビリーフ** = 思い込み（例: 「子供が好きなだけでは続かない」）
  - 背景には **家族（母親・夫）の影響** がしばしばある
- **アンビバレンス** = あれもこれもやりたい状態
  - → **エドガー・シャインのキャリアアンカー分析** が有効
- **転機** … リフレーミングの起点ワード

### 🌱 自己理解の深化
- **自己効力感（の低下）** … 問題視点の一つ
- **キャリアアンカー（エドガー・シャイン 8つ）**
  1. 専門・職能別コンピタンス
  2. 全般管理コンピタンス（経営管理）
  3. 自律・独立
  4. 保障・安定
  5. 起業家的創造性
  6. 奉仕・社会貢献
  7. 純粋な挑戦
  8. 生活様式（ワークライフバランス）
  - **使い方**: 「最後まで譲れないもの」を問うことで自己概念の中核を炙り出す

### 🗣 言い換え技術
- 抽象的な訴え → **具体的キーワード** に変換すると論理展開が楽
  - 例: 「経験・能力育成のチャンスがない」→「**仕事の広がりがない**」

## 📝 設問別 頻出パターン＆チェックリスト

### 🔵 設問1（事例の状況描写）
**チェックリスト** ✅
- [ ] 事柄（What / Who / When）が書けているか
- [ ] **感情描写**（不安／焦り／自信喪失／葛藤／モチベ低下／会社のために頑張りたい等）が入っているか
- [ ] 相談者の**価値観・想い**に触れているか
- [ ] 状況の**構造化**（相談者自身／周囲環境／予定タスク／家族の期待値 等）ができているか

> ⚠️ **3回連続で感情描写不足の指摘あり（4/25・4/26・5/2）**

### 🟢 設問2（CC応答の意図）
**質問意図のバリエーション**（複数を書き分けられるように）
- 課題の**明確化／確認**（※「特定」は強すぎる）
- **自己理解**を深める
- 相談者の**思い・具体的な考え**を確認
- 相談者の**気持ちへの共感**などCCの態度
- 取り組みの確認 → 姿勢／事柄の**承認** → **ラポール形成**
- 不安について**具体的な要因**を確認
- 「**確認する意図**」という表現も使えるように

### 🟡 設問3（問題の特定）
**問題視点のレパートリー**
- 基本3点: **仕事理解不足／自己理解不足／コミュニケーション不足**
- 拡張: **情報不足／状況把握不足／自己効力感の低下／ライフプラン不足**
- 構造: 大きな問題を **チャンク化** して捉える
- 背景要因: 家族（母親・夫）の影響、イラショナルビリーフ
- ⚠️ パターン化しないこと（同じ3点で済ませない）

### 🔴 設問4（今後の方針・方策）
**方策の型**
- 「**一緒に〜を考え、納得して〜できるよう支援**」型
- 中長期キャリアプランの**共同作成**支援
- ツール活用（**ジョブカード／モチベーショングラフ／VPI** 等）
- リフレーミング（ネガティブ → 転機・成長機会）
- 家族（夫等）との**相談提案**
- 自己理解不足 → **価値観を深めながら一緒に考える**
- ライフプラン不足 → 将来費用・サポートを具体化、ライフプラン作成

**クロージング表現**
- **「相談者が前向きに仕事を進められるよう支援していく」**
